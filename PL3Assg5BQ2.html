Problem statement: Write a code in any programing language that counts digits in a number

Code

int n,count=0;
cout << "enter a number";
cin >> n;
while(n>0)
{
n=n/10;
count++;
}
cout << "number of digits =" << count;

Sample input

3456

Sample output

Nunmber of digits= 4

Variables used in the program

n, count are the variables usedin the propgram

Time complexity

time complexity is O(log10n) as the loop iterates number of digit times. Therefore n can be given as 10d-1<=n < 10d , d is number of digits taking log on both sides , d-1 <= log n < d ,
